CODE HOUR

"Talk is cheap. Show me the code."

Rules and Regulations

GUIDELINES:

  • You have to complete all the tasks within 1 hour. In case no team is able to solve all the tasks in the given time, an entended time of 15 minutes will be provided to all.
  • Time is a critical point in this competition. try to complete the tasks in the least possible time.
  • All the tasks are to be submitted together in a folder. Once you submit the tasks, you will not be able to make any changes.
  • In case the source code submitted by the first team passes all the test cases, no other team's code will be tested.
  • The participant found using mobile (without permission) or internet will be eliminated on the spot.
  • The coordinators has the right to kick you out of the competition if you misbehave or create disturbance.
  • In case of any discrepency, the decision of the organisors will be final.


CODE CHALLENGE - THE TWIST:
  • You are LEVEL 2 coders. If you complete all the tasks before the LEVEL 1 winning team, you will have the opportunity to challenge them and win their prize money as well.
  • You will have to complete all tasks of LEVEL 1 within 60% of the time taken by the LEVEL 1 winners.
  • The LEVEL 1 winners may have completed their challenge before you guys. In that case, they will have the authority to challenge the LEVEL 1 winners. They will have to complete any 4/5 tasks of LEVEL 2 within the time taken by the LEVEL 2 winners.


CASH PRIZES:
  • The winner of first challenge will win Rs 375.
  • If you challenge LEVEL 1 winners and win, you will get Rs 1000.
  • If you challenge LEVEL 1 winners and loose, you will get 0 and the LEVEL 1 winners will get Rs 750.

Assignment 5

GST CALCULATOR

You have to develop a POINT-OF-SALES system to which you have to pass the ID of the items to be added in the bill. The products are divided into 4 categories each having different tax-slabs [5%, 12%, 18%, 28%].
Add a few products in each category by yourself where each product must have 4 properties:

  • ID
  • NAME
  • PRICE
  • TAX-RATE
The INPUT should include the number of items and ID of each item.
The OUTPUT must display the bill with 5 columns:
ID - NAME - PRICE - GST - TOTAL AMOUNT
At the end of the list, display the total amount to be paid by the customer.

NOTE: If any ID is passed against which no product exists, such products should be mentioned at the end of the list as UNTRACKED PRODUCTS.
